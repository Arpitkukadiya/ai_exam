Sets & Solutions

SET-1 Questions & Answers
1. Write a program to calculate square and cube of a given number.
square_cube(N, S, C) :- S is N*N, C is N*N*N.
2. Write a program to find maximum of three numbers.
max3(A,B,C,Max) :- Max is max(A, max(B,C)).
3. Write a program to calculate sum of 1 to N.
sum_n(1,1).
sum_n(N,Sum) :- N > 1, N1 is N-1, sum_n(N1,Sum1), Sum is Sum1 + N.
SET-2 Questions & Answers
1. Write a program to calculate area and perimeter of circle.
circle(R, Area, Perimeter) :- Area is 3.14*R*R, Perimeter is 2*3.14*R.
2. Write a program to check whether given number is divisible by 4.
divisible_by_4(N) :- 0 is N mod 4.
3. Write a program to find sum of numbers between N1 and N2.
sum_range(N1,N1,N1).
sum_range(N1,N2,Sum) :- N1 < N2, N3 is N2-1, sum_range(N1,N3,Sum1), Sum is Sum1 + N2.
SET-3 Questions & Answers
1. Write a program to convert year into minutes.
year_to_minutes(Y, M) :- M is Y * 365 * 24 * 60.
2. Write a program to check whether given number is odd or even.
is_even(N) :- 0 is N mod 2.
is_odd(N) :- 1 is N mod 2.
3. Write a program to print series from 1 to N.
series(N) :- series_helper(1,N).
series_helper(I,N) :- I =< N, write(I), nl, I1 is I+1, series_helper(I1,N).
series_helper(I,N) :- I > N.
SET-4 Questions & Answers
1. Write a program to calculate area and perimeter of circle.
circle(R, Area, Perimeter) :- Area is 3.14*R*R, Perimeter is 2*3.14*R.
2. Write a program to check whether student is pass or fail.
result(Per,'Pass') :- Per >= 40.
result(Per,'Fail') :- Per < 40.
3. Write a program to print series from N to 1.
series(N) :- N > 0, write(N), nl, N1 is N-1, series(N1).
series(0).
4. Write a program to print table of 5.
tables :-
    write('Enter Number for print table:'),
    read(N),
    between(1,10,X),
    A is N*X,
    write(N),write('*'),write(X),write('='),write(A),nl,
    fail;
    true.
